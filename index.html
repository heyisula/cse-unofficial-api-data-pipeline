CSE Unofficial API

Documentation for the unofficial Colombo Stock Exchange (CSE) API.

Forked from GH0STH4CKER/Colombo-Stock-Exchange-CSE-API-Documentation. Special thanks to the original author for the initial reverse engineering.

Disclaimer: This is a third-party documentation project and isn't affiliated with the Colombo Stock Exchange. Use these endpoints responsibly and at your own risk. This documentation is based on public analysis of network traffic.

Base URL

https://www.cse.lk/api/

Authentication & Headers

Most endpoints are public but it is recommended to mimic browser headers.

Content-Type: application/x-www-form-urlencoded
Origin: https://www.cse.lk
Referer: https://www.cse.lk/

Company Info Summary

POST /companyInfoSummery
Parameter Description
symbol
string		 The stock symbol (e.g., LOLC.N0000). 
{
  "reqSymbolInfo": {
    "symbol": "LOLC.N0000",
    "name": "L O L C HOLDINGS PLC"
  }
}

Trade Summary

POST /tradeSummary 
{"reqTradeSummery": [{"symbol": "ABAN.N0000", "price": 579.75}]

Detailed Trades

POST /detailedTrades
Parameter Description
symbol
string (optional)		 Filter by stock symbol. 
{
  "reqDetailTrades": [
    {
      "symbol": "ABAN.N0000",
      "price": 585.0
    }
  ]
}
